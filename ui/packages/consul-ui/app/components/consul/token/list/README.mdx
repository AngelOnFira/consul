# Consul::Token::List

A presentational component for rendering Consul ACL tokens. Note the `@token`
argument is used to highlight a certain token, so you'll mostly want to pass
the users current token in here, which will therefore be highlighted as the
your current token.

```hbs preview-template
<DataSource @src={{uri '/partition/default/dc-1/tokens'}} as |source|>
  <Consul::Token::List
    @items={{source.data}}
    @token={{hash
      AccessorID='00000000-0000-0000-0000-000000000002'
    }}
    @onuse={{noop}}
    @ondelete={{noop}}
    @onlogout={{noop}}
    @onclone={{noop}}
  />
</DataSource>
```

## Arguments

| Argument/Attribute | Type | Default | Description |
| --- | --- | --- | --- |
| `items` | `array` |  | An array of ACL tokens |
| `token` | `Token` |  | A token object to use for comparision for current token, usually the users current token |
| `onuse` | `function` |  | An action to execute when the `Use` action is clicked |
| `ondelete` | `function` |  | An action to execute when the `Delete` action is clicked |
| `onlogout` | `function` |  | An action to execute when the `Logout` action is clicked |
| `onclone` | `function` |  | An action to execute when the `Clone/Duplicate` action is clicked |
